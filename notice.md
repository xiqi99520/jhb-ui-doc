# Alert 警告

用于页面中展示重要的提示信息

## 基础用法

```html
	<JHB-alert title="成功提示的文案" type="success" show-icon></JHB-alert>
	<JHB-alert title="消息提示的文案" type="default" show-icon></JHB-alert>
	<JHB-alert title="消息提示的文案" type="info" show-icon> </JHB-alert>
	<JHB-alert title="警告提示的文案" type="warning" show-icon></JHB-alert>
	<JHB-alert title="错误提示的文案" type="error" show-icon></JHB-alert>
```

## 带文案描述

```html
	<JHB-alert title="成功提示的文案" type="success" description="文字说明文字说明文字说明文字说明文字说明文字说明" show-icon></JHB-alert>

	<JHB-alert title="消息提示的文案" type="default" description="文字说明文字说明文字说明文字说明文字说明文字说明" show-icon></JHB-alert>

	<JHB-alert title="消息提示的文案" type="info" description="文字说明文字说明文字说明文字说明文字说明文字说明" show-icon></JHB-alert>

	<JHB-alert title="警告提示的文案" type="warning" description="文字说明文字说明文字说明文字说明文字说明文字说明" show-icon></JHB-alert>

	<JHB-alert title="错误提示的文案" type="error" description="文字说明文字说明文字说明文字说明文字说明文字说明" show-icon></JHB-alert>
```
