# Tag 标签

用于标记和选择

## 基础用法

```html
	<JHB-tag type="" effect="plain">标签一</JHB-tag>
	<JHB-tag type="success" effect="plain">标签二</JHB-tag>
	<JHB-tag type="info" effect="plain">标签三</JHB-tag>
	<JHB-tag type="danger" effect="plain">标签四</JHB-tag>
	<JHB-tag type="warning" effect="plain">标签五</JHB-tag>
	<JHB-tag type="other" effect="plain">其他</JHB-tag>
```

## 不同主题

```html
	<JHB-tag type="" effect="dark">标签一</JHB-tag>
	<JHB-tag type="success" effect="dark">标签二</JHB-tag>
	<JHB-tag type="info" effect="dark">标签三</JHB-tag>
	<JHB-tag type="danger" effect="dark">标签四</JHB-tag>
	<JHB-tag type="warning" effect="dark">标签五</JHB-tag>
	<JHB-tag type="other" effect="dark">其他</JHB-tag>
```

## 可移除标签

```html
	<JHB-tag type="" closable>标签一</JHB-tag>
	<JHB-tag type="success" closable>标签二</JHB-tag>
	<JHB-tag type="info" closable>标签三</JHB-tag>
	<JHB-tag type="danger" closable>标签四</JHB-tag>
	<JHB-tag type="warning" closable>标签五</JHB-tag>
	<JHB-tag type="other" closable>其他</JHB-tag>
```
