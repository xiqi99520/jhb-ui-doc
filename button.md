# Button 按钮

常用的操作按钮

## 基础用法

基础的按钮用法

```html
	<JHB-button baseStyle="Primary">主要按钮</JHB-button>
	<JHB-button baseStyle="Stroke">主要按钮</JHB-button>
	<JHB-button baseStyle="Secondary">主要按钮</JHB-button>
	<JHB-button baseStyle="Primary" icon="el-icon-dianzan">主要按钮</JHB-button>
	<JHB-button baseStyle="Primary" rightIcon="el-icon-dianzan">主要按钮</JHB-button>
```
